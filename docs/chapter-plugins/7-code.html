7. 代码块添加行号和复制按钮

code 代码添加行号&复制按钮

这个插件可以为代码块添加行号和复制按钮，单行代码情况无行号。

如果需要去除代码复制按钮，可在配置文件进行配置 copyButtons 属性为 false。

配置使用方法：

{
    "plugins" : [ 
            "code" 
     ],
    "pluginsConfig": {
      "code": {
        "copyButtons": false
      }
    }
}

插件 Github 地址：https://github.com/TGhoul/gitbook-plugin-code

效果预览：

{
    1,
    2,
    3,
    ...
}
